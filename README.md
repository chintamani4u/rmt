# Building a RESTful API in Node and Express for RM Tool

Using the new Express 4.0 Router to build an API

This code is based on this blog.
[Read the tutorial](http://scotch.io/tutorials/javascript/build-a-restful-api-using-node-and-express-4)

## Requirements

- Node and npm

## Installation

- Clone the repo: `git clone git@github.com:chintamani4u/rmt.git`
- Install dependencies: `npm install`
- Start the server: `node server.js`

## Testing the API
Test your API using [Postman](https://chrome.google.com/webstore/detail/postman-rest-client-packa/fhbjgbiflinjbdggehcddcbncdddomop)
